# Logical-Model Golden Data

This directory contains frozen logical-model inputs used by the golden test
suite. Do not replace these files without validating their provenance,
licensing, checksums, and expected outputs.

Model of cell-fate decisions following death-receptor engagement.

- Source: BioModels `MODEL0912180000`
- URL: https://www.biomodels.org/MODEL0912180000
